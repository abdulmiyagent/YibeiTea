5. Technische & Organisatorische Maatregelen

5.1 Technisch

  • Encryptie: HTTPS/TLS voor alle communicatie
  • Wachtwoorden: bcrypt hashing met salt
  • Database: Row Level Security (RLS) in Supabase
  • Authenticatie: NextAuth.js met JWT tokens
  • API: Rate limiting en input validatie

5.2 Organisatorisch

  • Toegang: Beperkt tot noodzakelijk personeel
  • DPA's: Verwerkersovereenkomsten met alle verwerkers
  • Updates: Regelmatige security updates
  • Backup: Dagelijkse database backups
